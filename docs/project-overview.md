# Smart City AI Platform

## Overview
The Smart City AI Platform predicts traffic congestion using machine learning and visualizes city traffic data in an interactive dashboard.

## Key Features

- Traffic congestion prediction
- Traffic analytics visualization
- Interactive city map
- Heatmap for congestion monitoring
- Future traffic forecasting using ML models

## Technologies Used

Backend
- Python
- Flask

Frontend
- HTML
- JavaScript
- Chart.js
- Leaflet

Machine Learning
- Scikit-learn
- Linear Regression
